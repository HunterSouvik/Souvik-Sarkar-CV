# CV - Souvik Sarkar

A high-fidelity, single-page professional portfolio featuring modern aesthetics, cinematic animations, and a glassmorphic design system.

## Project Structure

This project is designed as a self-contained, high-performance single-page application (SPA) where all styling, structure, and behavior are optimized within a single file for rapid loading and ease of deployment.

```text
/
├── assets/          # Professional photos (Profile, The LaLiT) and Resume (PDF)
├── index.html       # Main Portfolio Hub (HTML, Tailwind CSS, and JS engine)
├── LICENSE          # License information
└── README.md        # Documentation
```

## Features

- **High-Fidelity UI**: Implements a unique "Cyber-Nebula" dark aesthetic using Glassmorphism, Tailwind CSS, and ambient glows.
- **Cinematic Animations**: Smooth scroll reveals, dynamic counters, and an interactive typewriter effect.
- **Interactive Elements**: Fully responsive mobile menu, back-to-top shortcut, contact form submission handling, and a canvas-based particle constellation animation.
- **Premium Assets**: High-quality visual representation of professional milestones, qualifications, and core technical systems.
- **Fully Responsive**: Meticulously crafted for seamless experiences across desktop, tablet, and mobile devices.

## How to Preview

Simply open `index.html` in any modern web browser or serve it using a local live server (e.g., VS Code's "Live Server" extension).

## How to Customize

- **Structure & Content**: Update professional text, contact details, work experience, and educational background within the semantic HTML sections of `index.html`.
- **Styles & Custom Colors**: Fine-tune custom glassmorphic properties, keyframe animations, and custom colors inside the `<style>` block of `index.html`.
- **Logic & Particles**: Adjust typewriter behaviors, counter delays, scroll triggers, or canvas particle densities inside the `<script>` block at the bottom of `index.html`.
